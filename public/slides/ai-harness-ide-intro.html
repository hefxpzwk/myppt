01 / 08 PROJECT INTRO

AI Harness IDE
복잡한 AI 구축을
작업 가능한 흐름으로

우리 프로젝트는 프롬프트, 모델 선택, 도구 연결, 평가, 배포를 하나의 IDE 안에서 설계하고 실행하는 AI Harness 전용 개발 환경입니다.

핵심 목표는 "잘 쓰는 AI"가 아니라 "재현 가능한 AI 시스템"입니다.

Mission

3x Faster

실험-검증-배포 리드타임 단축

02 / 08 PROBLEM

왜 지금 필요한가

  • AI 앱 개발은 코드, 프롬프트, 벡터DB, 에이전트 로직이 분산되어 관리됩니다.
  • 성능 저하가 생겨도 어떤 변경이 원인인지 추적하기 어렵습니다.
  • 팀 협업 시 실험 기록이 남지 않아 동일한 실패를 반복합니다.
  • 결과적으로 PoC는 빠르지만 프로덕션 전환은 느립니다.
03 / 08 SOLUTION

AI Harness IDE가 제공하는 것

Visual Harness Builder

모델, 툴, 메모리, 평가 노드를 연결해 실행 흐름을 시각적으로 구성

Prompt + Code Co-Edit

프롬프트와 앱 코드를 같은 워크스페이스에서 버전 관리

Eval-first Workflow

테스트셋/메트릭 기반으로 품질 변화 자동 리포트

04 / 08 CORE FEATURES

핵심 기능

1. 템플릿 시작

RAG, Agent, Chatbot 템플릿으로 빠른 시작

2. 실시간 트레이스

토큰 사용량, 지연 시간, 에러 포인트를 단계별 추적

3. 데이터셋 버전

실험에 사용한 문서/질문셋을 자동 스냅샷

4. 팀 리뷰

변경점 diff와 코멘트 기반 협업

5. 원클릭 배포

검증 통과 Harness만 스테이징/프로덕션 배포

05 / 08 ARCHITECTURE

기술 아키텍처

Editor Layer

프로젝트 구조/노드 그래프/프롬프트 편집기

Runtime Layer

모델 라우팅, Tool Calling, Guardrail, 캐시 처리

Observability Layer

로그, 평가, 비용 분석, 실험 리플레이

06 / 08 TARGET USERS

누가 가장 크게 이득을 보는가

  • 사내 AI 기능을 빠르게 제품에 붙여야 하는 스타트업 개발팀
  • PoC 이후 품질 관리와 재현성 문제를 겪는 AI 엔지니어 조직
  • 비개발 직군과 함께 프롬프트/실험 협업이 필요한 PM 조직
